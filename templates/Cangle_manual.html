Cangle Manual

Available Query Commands:

  • Creating an Event
  • Obtaining Course ID
  • Course Events & Assignments Integration
    • All Assignments (/ Events) for the Course
    • All Assignments (/ Events) within a Specified Time Window
    • Specific Assignment (/ Event) Integration

Command Formats:

For all curl commands if you wish to indicate a space please replace the space with %20. If not curl will indicate a Response 400 Error has occurred.

Creating an Event

command = create

Please also specify the event_name, start (format: YYYY-MM-DD) and end (format: YYYY-MM-DD).

Note: The created event will be reflected in you personal Canvas Calendar and Google Calendar.

Obtaining Course ID

command = course_id

Please specify the course_name (should be the same as it appears on your canvas page).

Note: The course id of the course specified will be returned in Terminal or Browser.

Course Events & Assignment Integration

Default event type is assignment. If you wish to change event type please specify type = assignment or event.

command = integrate

To integrate all events for a particular course, specify all_events=1 and the course_id.

To integrated course events within a specific window, specify the following fields: course_id, start (format: YYYY-MM-DD), and end (format: YYYY-MM-DD).

To integrated one specific event, specify the course_id and event_name (as it appear on Canvas).

Note: The integrated event(s) will appear on your Google Calendar. Unless program indicates that something has gone wrong.
