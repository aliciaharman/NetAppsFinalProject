Cangle Manual

Available Query Commands:

  • Creating an Event
  • Obtaining Course ID
  • Course Events & Assignments Integration
    • All Assignments (/ Events) for the Course
    • All Assignments (/ Events) within a Specified Time Window
    • Specific Assignment (/ Event) Integration

Command Formats:

Creating an Event

command = create

Please also specify the title, start_date, start_time, end_date, and end_time.

Obtaining Course ID

command = course_id

Please specify the course_name (should be the same as it appears on your canvas page).

Course Events & Assignment Integration

Default event type is assignment. If you wish to change event type please specify type = assignment or event.

command = integrate
